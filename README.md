# Machine Learning Project Portfolio
This repository contains machine learning and data science projects demonstrating data preprocessing, feature engineering, predictive modeling, and performance evaluation using Python.

## Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Projects

### Loan Approval Prediction
Machine learning model to predict loan approval using classification algorithms such as Logistic Regression, Random Forest, and SVM.

### Credit Risk Prediction
Predicts potential loan default risk based on applicant financial information.

### Customer Churn Prediction
Predicts whether a customer is likely to leave a service using classification models.

## Skills Demonstrated
- Data Cleaning
- Feature Engineering
- Machine Learning Modeling
- Model Evaluation
- Data Visualization
